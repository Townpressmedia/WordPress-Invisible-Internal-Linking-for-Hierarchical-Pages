# Invisible Internal Links for WordPress

This plugin automatically injects invisible, crawlable internal links between parent and sibling pages.

## Features
- No editor modification
- SEO-safe (no display:none)
- Improves crawl paths and topical clustering
- Lightweight and cache-friendly

## Installation
1. Upload the plugin folder or place the PHP file in /wp-content/mu-plugins/
2. Add the CSS to your theme or global stylesheet

## Use Cases
- Venue directories
- Location-based service pages
- Hierarchical content hubs
