# Invisible Internal Links for WordPress (v1.1)

Automatically injects invisible, crawlable internal links between
parent and sibling pages to strengthen SEO and site architecture.

## What's New in v1.1
- Transient caching for near-zero runtime cost
- Safety limit on sibling links (default: 50)
- Improved inline documentation

## Features
- No editor modification required
- SEO-safe hiding technique (no display:none)
- Improves crawl depth and topical clustering
- Works with any hierarchical page structure

## Installation
### Option 1: MU-Plugin (Recommended)
Place `invisible-internal-links.php` into:
`/wp-content/mu-plugins/`

### Option 2: Standard Plugin
Upload to `/wp-content/plugins/` and activate.

## Required CSS
Add `seo-internal-links.css` to your global stylesheet.
